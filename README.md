# SailFish SDK

## SDK简介
 SailFish SDK可以帮助第三方网站及应用快速构建一个完善的客服系统生态圈，SDK提供较为完善的客服应用功能，及简洁的API接口。

### 流程图

sdk通信
```
    sdk --> client --> sdk
```

sdk组成
```
    - sdk
      - manage      // 数据模型
      - dom         // Dom操作
      - util        // 方法集合
      - ajax        // 请求
      - message     // 通信
```
